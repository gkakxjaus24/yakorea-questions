📋 위젯 설치 방법

아래 스크립트 태그를 홈페이지 </body> 직전에 추가하세요:

<script src="http://localhost:3001/widget/chat-widget.js"></script>

🧪 테스트 시나리오

1️⃣ 위젯 버튼 클릭 → 채팅창 열기
2️⃣ 언어 탭 전환 (한국어/영어/중국어/일본어)
3️⃣ ⌨️ 버튼 클릭 → 가상 키보드 열기/닫기
4️⃣ 자동응답 테스트 메시지 전송:
• "체크인 언제예요?" → 자동응답 기대
• "와이파이 비밀번호 뭐예요?" → 와이파이 정보 응답 기대
• "아무말이나 해도 되는 이상한 질문" → 매니저 연결 유도 기대
5️⃣ "매니저와 대화하기" 버튼 클릭 테스트

⚠️ 서버가 http://localhost:3001 에서 실행 중이어야 합니다.
